# Screenshots for Framer Marketplace Submission

## Required Screenshots (5 recommended)

To complete the marketplace submission, capture the following screenshots in Framer:

### 1. Component List View
**Filename**: `01-component-list.png`
**What to show**:
- Plugin UI with multiple components selected
- Display icons, badges (MASTER/INSTANCE), and component details
- Show All/None selection buttons and counter
- Highlight the clean, native Framer design token styling

**How to capture**:
1. Open Framer project with several components
2. Select 5-7 frames/components on canvas
3. Open Scheemer plugin
4. Take screenshot of full plugin panel

---

### 2. Component Editing Mode
**Filename**: `02-component-editing-mode.png`
**What to show**:
- Plugin detecting component master automatically
- Shows message like "Button master + 3 selected"
- Demonstrates component mode support

**How to capture**:
1. Edit a component (double-click to enter editing mode)
2. Select additional variants/frames inside
3. Open Scheemer plugin
4. Take screenshot showing master auto-detection

---

### 3. Export Dialog / Selection
**Filename**: `03-export-selection.png`
**What to show**:
- Multiple items checked for export
- Download button highlighted
- Shows selection counter (e.g., "5/12")
- Different component types (frames, instances, masters)

**How to capture**:
1. Select mix of components
2. Check 5-7 items in plugin
3. Hover over Download button
4. Take screenshot of full plugin panel

---

### 4. JSON Output Example
**Filename**: `04-json-output.png`
**What to show**:
- Sample JSON file open in code editor (VS Code, Sublime, etc.)
- Shows clean, structured export format
- Highlights nested children structure
- Display both structure and properties sections

**How to capture**:
1. Export a component from Scheemer
2. Open downloaded JSON in code editor
3. Expand structure and children arrays
4. Take screenshot showing full JSON structure

---

### 5. Batch Export Workflow
**Filename**: `05-batch-export-workflow.png`
**What to show**:
- Finder/Explorer window showing multiple downloaded JSON files
- Demonstrates sanitized filenames (lowercase, hyphens)
- Shows real-world usage with design system components
- Optional: Side-by-side of Framer components + exported JSONs

**How to capture**:
1. Export 5-10 components from Scheemer
2. Open Downloads folder or destination
3. Arrange files to show filenames clearly
4. Take screenshot

---

## Technical Requirements

- **Format**: PNG (preferred) or JPG
- **Resolution**: Minimum 1280x800, recommend 1920x1080 for high-DPI displays
- **Color**: RGB color space
- **Quality**: High quality, sharp text, no compression artifacts
- **File size**: Under 5MB per image

## Design Guidelines (from Framer docs)

- Sharp, high-quality assets
- Show plugin and its UI
- Keep minimal - avoid clutter
- No additional text or labels overlaid
- Make visually engaging
- Use original, properly licensed content

## Optional: Annotated Versions

Consider creating annotated versions for GitHub README that highlight key features with arrows/callouts, but submit clean versions to Framer marketplace.

---

## Submission Checklist

Before submitting to Framer marketplace:
- [ ] All 5 screenshots captured
- [ ] Images are high resolution (1280x800+)
- [ ] File sizes under 5MB each
- [ ] No sensitive/private data visible
- [ ] Plugin UI clearly visible
- [ ] Screenshots show actual functionality (not mockups)
- [ ] Filenames are descriptive
- [ ] Images saved in this directory
